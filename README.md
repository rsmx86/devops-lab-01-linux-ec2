\# 🚀 DevOps Lab 01 - EC2 + Nginx



\## 📌 Objetivo

Este projeto demonstra a criação de uma infraestrutura básica na AWS utilizando EC2, com instalação automatizada do Nginx via script.



\---



\## 🧠 O que foi feito



\- Criação de instância EC2 (AWS)

\- Acesso via SSH com chave .pem

\- Instalação do Nginx

\- Deploy de página web

\- Automação com script Bash



\---



\## ⚙️ Script



O arquivo `setup.sh` realiza:



\- Atualização de pacotes

\- Instalação do Nginx

\- Criação de página HTML

\- Inicialização do serviço



\---



\## 🚀 Como executar



```bash

chmod +x setup.sh

./setup.sh
